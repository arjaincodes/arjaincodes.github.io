Chest X-ray Medical Diagnosis with Deep Learning

In this project I built a deep learning model that can interpret chest X-rays to classify different disease causes.
It is a neural network to diagnose whether or not a patient has pneumonia given an image of a chest X-ray
Learnings from the project:

  • Data preparation
  • Visualizing data
  • Preventing data leakage
  • Model Development
  • Addressing class imbalance
  • Leveraging pre-trained models using transfer learning
  • Evaluation
  • AUC and ROC curves
